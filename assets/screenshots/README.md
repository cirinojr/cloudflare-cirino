# Screenshots

Place repository screenshots in this folder so they render in GitHub and keep the README visual section consistent.

Expected filenames:

1. `01-settings-overview.png`
2. `02-connection-test-status.png`
3. `03-recent-activity-log.png`
4. `04-targeted-vs-everything.png`

Capture guidance:

- Use the default WordPress admin theme.
- Use realistic but redacted values for Zone ID and credentials.
- Keep browser zoom at 100% and crop only obvious personal data.
- Prefer PNG with width around 1600px for readable details.
